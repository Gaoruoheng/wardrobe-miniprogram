# Kuma Closet 衣柜管理小程序

面向喜欢整理穿搭、需要多人协作管理衣物的个人用户、情侣、室友和家庭用户，这是一款可爱风格的微信小程序衣柜管理工具。它支持用户创建独立云端衣柜、分类记录衣服、上传图片、快速搜索、拖动排序、编辑衣服状态，并能通过共享衣柜让他人一起查看和操作。

## 主要功能

- 用户注册 / 登录，支持 30 天免登录
- 不同用户拥有独立衣柜
- 支持共享衣柜，其他用户可查看和操作
- 支持添加衣服、分类管理、图片上传和衣服编辑
- 首页支持实时搜索、分类跳转和拖动排序
- 支持衣服状态：在库、使用中、已收纳
- 支持图片压缩、缩略图和云端存储
- 支持页面缓存，提高二次进入速度

## 页面结构

- `pages/home/home`：登录、注册、衣柜入口
- `pages/index/index`：衣柜首页
- `pages/add/add`：添加衣服
- `pages/manage/manage`：衣柜管理
- `pages/category-detail/category`：分类详情
- `pages/item-detail/item-detail`：衣服编辑

## 技术方向

- 微信小程序原生开发
- 微信云开发数据库
- 微信云存储
- 云函数处理关键写操作
- 本地缓存优化页面加载速度

## 项目目标

让用户可以更轻松地管理衣柜、记录衣服、安排穿搭，并且在换设备后仍然能看到自己的完整数据。
